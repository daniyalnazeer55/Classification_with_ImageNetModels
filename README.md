# Image Loading and Visualization with TensorFlow and Matplotlib

This project contains a simple Jupyter Notebook (`.ipynb`) that illustrates how to locate image files using glob patterns, read them into memory, and display them in a structured subplot layout using Python.

## Prerequisites

Ensure you have Python 3 installed on your system. You will also need to install the required Python packages listed below.

## Installation & Requirements

1. Clone or download this project directory.
2. Install the required dependencies by running:

pip install -r requirements.txt
